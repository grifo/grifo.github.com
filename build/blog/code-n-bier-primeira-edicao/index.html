01 Aug 2012

Na noite de ontem (01.08.2012), rolou a primeira edição do Code n' Bier. Um projeto que tem como intenção levar programação e pessoas para dentro de um bar com o objetivo de promover aprendizado, integração e diversão.

As fotos do evento já estão na página da Grifo no Facebook.

Dojo

A atração principal desta primeira edição foi um Dojo. Se você ainda não teve contato com Dojos, confira a apresentação inicial.

Nosso problema era construir uma lista de múltiplos. A definição completa, resolução e frameworks de teste utilizados já estão no repositório git do evento, confira.

Como é de praxe em todo Dojo, vamos a retrospectiva:

Desafios e Sorteios

No fim do Dojo a galera foi desafiada a surpreender a platéia em cinco minutos valendo um ingresso pro BrazilJS e uma caneca da Grifo.

As ideias que surgiram de desafio foram as seguintes:

  • fizzbuzz
  • layer centralizada
  • project euler em node
  • circulos com canvas
  • circulo com divs de 1x1px
  • sistema operacional em js
  • fibonacci

Os primeiros dois desafiados tentaram programar no interpretador do Node e não conseguiram finalizar o desafio.

A platéia resolveu dar uma segunda chance para eles e @jmurowaniecki resolveu fibonacci e @n2liquid fez o famoso fizzbuzz e emplacaram o desafio.
